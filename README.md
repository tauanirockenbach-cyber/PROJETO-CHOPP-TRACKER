# PROJETO-CHOPP-TRACKER
Uma cervejaria artesanal de médio porte colabora com o SENAI para melhorar o envase de chopp em barris retornáveis. Enfrentando desafios como controle operacional, rastreabilidade de lotes e redução de perdas, a empresa busca entender tecnologias, tendências e oportunidades de melhoria, considerando diversas restrições.
